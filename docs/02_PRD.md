# Forever Alive

Product Requirement Document

Version 1.0

---

# Product Overview

Forever Alive allows users to prepare
messages in advance
and automatically deliver them
to their loved ones
when they can no longer do so themselves.

# Core Feature

Safety Check

Users periodically confirm they are active.

If confirmation is missed,

the system starts a confirmation process.

Only after all verification steps
are completed,

messages are delivered.
